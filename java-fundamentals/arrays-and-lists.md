# 配列とリスト

Java では固定長の `array` と可変長の `List` を使い分けます。

対象コードは [SumCalculator.java](/C:/apps/learn-java-with-tests/src/main/java/dev/learnjavawithtests/arrays/v1/SumCalculator.java)、
テストは [SumCalculatorTest.java](/C:/apps/learn-java-with-tests/src/test/java/dev/learnjavawithtests/arrays/v1/SumCalculatorTest.java) にあります。

## 学ぶこと

- 配列の走査
- `List.of(...)`
- 複数ケースをまとめて検証するテスト

Go 版の「arrays and slices」に相当する入り口として、Java では `int[]` と `List<int[]>` を扱っています。
